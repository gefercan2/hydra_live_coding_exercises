# hydra_live_coding_exercises
Hydra is live code-able video synth and coding environment that runs directly in the browser. It is free and open-source and made for beginners and experts. Hydra makes it possible to easily create ‘patches’ in the browser, to achieve many different types of imagery. These patches are a series of JavaScript functions that are chained together to bend and break colors and shapes to make images to your liking.

There are 5 main types of functions that hydra has implemented to make these visuals possible; source, color, geometry, blend, modulate. These are the core components for creating super psychedelic and awesome visuals on the screen.
